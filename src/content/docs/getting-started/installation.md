## Installing UDS

You'll need to install the UDS binary in order to continue with this quickstart. Installation will vary based on your platform.

### macOS

The UDS binary is available via [Homebrew](https://brew.sh/), after adding the defenseunicorns tap:

```
brew tap defenseunicorns/tap && brew install uds
```
